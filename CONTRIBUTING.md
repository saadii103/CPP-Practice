# 🤝 Contributing to CPP Practice

Thank you for your interest in contributing to this repository 💻

This repository is created for practicing **C++ programming concepts** and solving different programming questions. Contributions that improve code quality, documentation, or add useful practice questions are always welcome 🚀

---

# 📌 How to Contribute

## 1️⃣ Fork the Repository

Create your own copy of this repository by clicking the **Fork** button on GitHub 🍴

---

## 2️⃣ Clone the Repository

```bash id="exz0wl"
git clone https://github.com/saadii103/CPP-Practice.git
```

---

## 3️⃣ Create a New Branch

```bash id="jzh2s9"
git checkout -b feature-name
```

---

## 4️⃣ Make Your Changes

You can contribute by:

* ➕ Adding new C++ practice questions
* 🛠️ Improving existing solutions
* 🐞 Fixing bugs
* 📖 Improving documentation
* ⚡ Optimizing algorithms
* ✨ Enhancing code readability

---

# 🧠 Coding Guidelines

Please follow these rules while contributing:

* ✅ Write clean and readable code
* ✅ Use meaningful variable names
* ✅ Add comments where necessary
* ✅ Maintain proper indentation
* ✅ Avoid unnecessary complexity

---

# 📝 Commit Guidelines

Use clear and meaningful commit messages.

### Example

```bash id="g9mjlwm"
git commit -m "Added new array practice question"
```

---

# 🚀 Push Changes

```bash id="vr4f3w"
git push origin feature-name
```

---

# 🔄 Create a Pull Request

After pushing your changes, create a Pull Request describing your improvements 📬
