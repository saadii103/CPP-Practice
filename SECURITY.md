# 📋 What to Include in Your Report

Please include the following information when reporting a vulnerability:

* 🐞 Description of the issue
* 📍 Location of the vulnerability
* ⚠️ Possible impact
* 🛠️ Suggested fix (if available)
* 📸 Screenshots or examples (optional)

# ⏱️ Response Time

All valid security reports will be reviewed as soon as possible. Appropriate fixes and updates will be made when necessary.
